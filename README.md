游戏特色：
挖矿盘面：采用 grid 网格布局，模拟真实的矿洞，点击未探索的方块进行挖掘。

随机事件：

矿石：从普通的石头到稀有的钻石，价值不同。

炸弹：挖到炸弹会扣除金币并产生眩晕效果（几秒内无法挖矿）。

惊喜宝箱：随机获得大量金币或回复状态。

装备系统：使用金币在商店购买更高级的镐子，高级镐子可以挖掘得更快，且有更高几率发现稀有矿石。

背包与经济：挖到的矿石存入背包，一键出售换取金币。

如何开始：
点击下方的“挖矿”区域中的土块。

积累矿石后，点击“出售矿石”换钱。

攒够钱去“工具商店”升级你的镐子。

玩法说明：
游戏目标：成为最富有的矿工，解锁“激光切割机”。

网格互动：

土块 (棕色)：未知区域，点击挖掘。

矿石：挖掘后显示，自动进入背包。

炸弹 (💣)：挖掘到后屏幕会震动，你会损失 10% 的金币并被眩晕 2 秒。

惊喜 (🎁)：直接获得随机数额的金币奖励。

策略：

初期不要攒太多矿石，尽快卖出（点击绿色按钮）换钱买第二把“铁镐”，因为它能稍微增加好运气。

如果不小心挖到炸弹，虽然会扣钱，但也是游戏的一部分！

高级工具不仅挖掘速度快（冷却时间短），而且有更高的 luck (幸运值)，能显著增加挖到钻石和宝箱的概率，减少挖到炸弹的概率。
